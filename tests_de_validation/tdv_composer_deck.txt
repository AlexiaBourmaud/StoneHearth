TV01: ComposerDeck nominal
Contexte: le joueur est connecter, il possède un deck "D1", et au moins les cartes basiques dans sa collection 

Entrées: deck="D1" et carte="B02"

Scénario: 
1) Le joueur selectionne "Modifier mon deck"
2) Le joueur selectionne le deck "D1"
3) Le joueur selectionne la carte "B02"
4) Le joueur termine l'interaction

Résultats Attendus: 
- La carte "B02" est ajouté au deck "D1"

Moyens validation:
- Visualisation du deck "D1" 
- Réaliser le test TV02 pour retirer la carte "B02" du deck "D1".

TV02: ComposerDeck Supression nominal
Contexte: Le joueur est connecté, il possède la carte "B02" dans son deck "D1". Ce test peu être réaliser après le test TV01.

Entrées: deck: "D1" et carte: "B02".

Scénario:
1) Le joueur choisit "composer votre deck".
2) Le joueur selectionne le deck "D1".
3) Le joueur selectionne le carte "B02" présente dans son deck "D1".
4) Le joueur met fin à l'interaction.

Résultats attendus:
- La carte "B02" n'appartient plus au deck "D1".

Moyens de validation:
- Afficher le deck "D1": la carte "B02" doit avoir disparu.
- Réaliser le test TV01 pour remettre la carte "B02" dans le deck "D1".

TV03: ComposerDeck AjoutErreur : plus de place dans le deck
Contexte: Le joueur est connecté, il possède un deck "D1" totalement remplit. Il possède la carte "B02" dans sa collection.

Entrées: deck: "D1" et carte: "B02".

Scénario:
1) Le joueur choisit "composer votre deck".
2) Le joueur selectionne le deck "D1".
3) Le joueur selectionne le carte "B02" présente dans son deck "D1".


Résultats attendus:
- Un message annonce "Il n'y a plus de place dans votre deck D1."
- La carte "B02" ne doit pas avoir été ajouté dans le deck "D1".
- L'écran affiche les cartes du deck "D1" et les cartes de la collection.

Moyens de validation:
- Visuel pour le message.
- Afficher le deck "D1": la carte "B02" ne doit pas être présente.
- Visuel: liste des cartes du deck D1 et liste des cartes de la collection.
